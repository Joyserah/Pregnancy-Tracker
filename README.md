\# Pregnancy Tracker App



A web application for tracking pregnancy progress and receiving personalized reminders for vaccines, lab work, management, and exercise.



\## Setup



```bash

pip install -r requirements.txt

python app.py

```



Visit http://localhost:5000



\## Features



\- Patient login/registration portal

\- Input pregnancy week (1-42)

\- Automatic reminders based on pregnancy stage:

&#x20; - Vaccines (Tdap, Flu)

&#x20; - Lab work (blood tests, ultrasounds, screenings)

&#x20; - Management tips (prenatal vitamins, kick counts)

&#x20; - Exercise recommendations (walking, yoga, swimming)



\## Usage



1\. Register a new account

2\. Login with credentials

3\. Enter current pregnancy week

4\. View personalized reminders for your stage



Patient data stored in `patients.json`
